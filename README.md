# task-master

A lightweight Flask-based task management web application that helps users create, update, and track their daily tasks efficiently.

📌 Features
Create new tasks
View all tasks
Update task status
Delete tasks
Simple and responsive UI using HTML templates
Lightweight Flask backend

🛠️ Tech Stack
Python 3.x
Flask
HTML, CSS (Jinja2 templates)
Docker (optional support)
