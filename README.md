# Task Master

A simple and efficient task management web application built using Flask. It allows users to create, update, and manage daily tasks with ease.

## Tech Stack
- Python
- Flask
- HTML, CSS (Jinja Templates)

## Features
- Create new tasks
- View all tasks
- Update task status
- Delete tasks
- Simple and responsive UI
- Lightweight Flask backend

## Setup
```bash
git clone https://github.com/Ahrar-1898/task-master.git
cd task-master
pip install -r requirements.txt
python app.py

Usage

Open your browser and go to:
http://127.0.0.1:5000
